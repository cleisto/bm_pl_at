BTC Power Law Monitor is a Bitcoin analysis app based on the mathematical Power Law model — a scale-invariant framework that has described Bitcoin's price trajectory since 2009.

THREE-LAYER MODEL:
• Layer 1 — Trend: Decay Power Law with floor, median, and ceiling bands
• Layer 2 — Oscillation: Log-periodic oscillation with amplitude decay for cycle timing
• Layer 3 — Macro: ISM Manufacturing PMI overlay for business cycle context

FEATURES:
• Live Bitcoin price via CoinGecko API
• Fair Value zone classification (Deep Value → Euphoria)
• Multi-year price projections with ±1σ prediction bands
• Cycle peak/trough timing with ±3 month accuracy
• DCA (Dollar Cost Averaging) multiplier recommendations
• Take-profit trigger levels
• Gold/BTC rotation signal
• Interactive charts with pinch-to-zoom
